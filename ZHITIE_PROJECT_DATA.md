## 8. 資料格式說明

### 8.1 日期格式
- 民國年格式：`115.01.02` = 民國115年1月2日 = 西元2026年1月2日
- 轉換公式：西元年 = 民國年 + 1911

### 8.2 金額格式
- 幣別：新台幣（NTD）
- 格式：整數（無小數點）

### 8.3 專案編號格式
- 格式：`PJ` + 4位數字
- 範例：PJ0001, PJ0002

### 8.4 供應商分類對應獎金計算

| 供應商分類 | 獎金計算歸屬 |

| 原物料 | 材料費 ✓ |

## 9. 獎金計算公式

```
專案總獎金額 = 訂單金額 - 佣金 - 專案成本
專案成本 = (材料費 + 人工費) × 1.15
```

**發放條件**：收款率 ≥ 80%
