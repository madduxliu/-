# 智鐵機械財務管理系統

## 版本: v1.1 (開發中)

## 核心公式
```
專案總獎金 = 訂單金額 - 佣金 - (材料費 + 人工費) × 1.15
```

## 功能模組
- [x] 儀表板 - 總覽統計
- [x] 應收帳款 - CRUD、收款記錄
- [x] 應付帳款 - CRUD、付款標記
- [x] 工程專案 - CRUD、成本管理
- [ ] 佣金管理 - 代理/轉介/回扣 (待完成)
- [x] 獎金結算 - 公式計算
- [x] 系統設定 - 公司/客戶/供應商
- [ ] 資料備份 - 匯入/匯出 (待完成)
